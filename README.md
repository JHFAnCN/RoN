# 指挥官技能（RoN_CommanderSkills）

## 本模组做什么

本模组为 **Reign of Nether RTS**（下界之治）增加一套 **指挥官技能** 系统。它是独立附加模组，必须先安装 Reign of Nether RTS 才能使用。

在 RTS 模式下选择阵营进行游戏时，屏幕左侧会出现技能面板。每个阵营有各自的技能：需要消耗食物、木材、矿物进行 **解锁** 与 **升级**，每次施放技能也会消耗 **使用代价**。解锁和升级还依赖你先建造指定建筑。技能施放后有 **冷却时间**。

**村民（灾厄）阵营：**
- **战斗号角**：在选定范围内为己方单位施加一段时间增益（攻击伤害与攻速、移速提升）。
- **休整**：在选定位置生成治疗区域，区域内己方单位持续回复生命。
- **坩埚狂欢**：在选定位置生成效果圈，圈内敌方/中立单位获得随机负面效果（如缓慢、虚弱），并可为己方/盟友施加增益。

**怪物（亡灵）阵营：**
- **不死者**：在一段时间内，全局己方/盟友的亡灵单位每秒回复生命（无需点击地图即可生效）。
- **幽匿护佑**：在选定范围内，己方单位不因白天燃烧，亡灵单位死亡时有概率复活。
- **幻翼特攻队**：选定一个圆形区域，延迟一段时间后在该区域上方召唤苦力怕单位落下。

**猪人阵营：**
- **高效传送门**：在一段时间内，己方所有生产单位的建筑训练单位速度提升（全局生效）。
- **悬赏**：在选定范围内标记敌方单位；被标记单位死亡时，你获得基于该技能使用代价的资源奖励。
- **错位传送**：先左键点击一个单位，再左键点击另一个单位（己方、敌方或中立均可），两者位置互换。

游戏中，每个技能与升级按钮都有悬浮提示，会说明消耗、冷却和建筑要求。使用方式：左键点击技能选中，再在地图上左键点击施放（或按界面提示操作）；右键取消当前选中。
