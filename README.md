# RoN Match — Project Page / 项目介绍

What This Mod Does
RoN Match is an extension mod for Reign of Nether RTS, adding the following features on top of the original:

Match / Preparation Lobby
Players can enter the preparation interface via commands or in-game methods. In the preparation page, they can select their team (A–Z or random), spawn point color, faction (Villagers / Monsters / Piglins / random), and click Ready. Once everyone is ready, authorized players can initiate Start Game, leading to a countdown. When the countdown ends, the match begins. During the countdown, any player can terminate the countdown by pressing the "Cancel Game" button or canceling their ready status.

Automated Handling at Match Start
When the match begins, the mod automatically places starting buildings (Town Center / Tomb / Central Portal) at each player's chosen spawn point, eliminating the need to go through the original mod's "select starting position" process. Teammates are automatically allied. Unit glow colors are displayed according to each player's chosen color, and the spectator view automatically focuses on each player's spawn point at the start.

Settings
Players with permissions can open the settings page to configure: population cap, whether countdown is enabled and its duration, friendly fire, and whether allied units can be controlled. The countdown is enabled by default, and mid-match joining is allowed by default.

Commands

rts-preparationhall — Opens the preparation lobby (can specify another player).

rts-kick <player name> — Kicks a player from the preparation lobby (requires permission).

rts-restmatch — Resets the match state (e.g., cleans up after a match ends; requires permission).

rts-match-settings — Opens match settings (requires permission).

Notes Before Downloading
Dependency: Reign of Nether RTS (by SoLegendary) must be installed first. This mod runs only on top of it and requires the same Minecraft and Forge versions as the base mod.

### 本模组做什么

RoN Match 是 **Reign of Nether RTS** 的 **扩展模组**，在原有基础上增加：

- **对局 / 准备大厅**  
  玩家通过指令或游戏内方式进入准备界面，在准备页中选择 **队伍**（A–Z 或随机）、**出生点颜色**、**阵营**（村民 / 怪物 / 猪灵 / 随机），并点击 **准备**。当所有人准备后，有权限的玩家可发起 **开始游戏**，进入倒计时；倒计时结束后对局开始。倒计时期间，任意玩家可通过「终止游戏」按钮或取消准备来 **终止倒计时**。

- **开局时的自动处理**  
  对局开始后，模组会 **自动在每位玩家所选出生点放置起始建筑**（城镇中心 / 陵墓 / 中央传送门），无需再走原模组的「选择起始位置」流程。**同队玩家会自动结盟**。单位发光颜色按玩家所选颜色显示；**上帝视角会在开局后自动对准各自出生点**。

- **设置**  
  拥有权限的玩家可打开设置页，配置：**人口上限**、**是否开启倒计时及倒计时秒数**、**友军伤害**、**是否可控制盟友单位**。默认开启倒计时，默认允许开局后中途加入。

- **指令**  
  - `rts-preparationhall` — 打开准备大厅（可指定其他玩家）。  
  - `rts-kick <玩家名>` — 将某玩家踢出准备大厅（需权限）。  
  - `rts-restmatch` — 重置对局状态（如对局结束后清理；需权限）。  
  - `rts-match-settings` — 打开匹配设置（需权限）。
